Packagewe3d.layer
Classpublic class Layer
InheritanceLayer Object

A Layer is used for 2d output. Every Layer have a Sprite and optional a Bitmap. By default the containers of the layer are created and attached to the root target of the View Multiple layers in a View can be sorted by the polygons the Layer contains A Layer can be added to one View only
