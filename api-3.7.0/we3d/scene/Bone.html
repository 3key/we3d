Classpublic class Bone
InheritanceBone Object3d flash.events.EventDispatcher

Bones can be added only to a SceneObjectBones object. A Bone is really simple, they know a set of points, wich will be transformed by the Bone. To animate a Bone, first set the restPosition to the initial position of the Bone. Then position and rotate the Bone to transform the points connected with the Bone.
