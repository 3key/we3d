setLayer()method 
public function setLayer(view_id:int, lyr:Layer):void

Set the 2d layer for the scene object. It's possible to set more layers for different views if multiple views are used with the same SceneObjects. Note: Objects with GPU enbaled have to be in the firstLayer (view.firstLayer) wich is the default Layer.

Parameters

view_id:int — the scene wich provides the layer
 
lyr:Layer — the layer were the object should be rendered
