Packagewe3d.scene
Classpublic class Scene3d
InheritanceScene3d Object

A Scene3d contains the SceneObjects and one Camera3d. Add or remove SceneObjects with the add and remove methods or use the objectList array directly. Call scene.invalidate() to tell a View that the scene need a redraw.
