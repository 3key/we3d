Packagewe3d.scene
Classpublic class SceneObjectMorph
InheritanceSceneObjectMorph SceneObject Object3d flash.events.EventDispatcher

Morph object with vertex map frames. For best render results, a morph should contain only triangles and two point polys. The MD2 file loader generates SceneObjectMorph objects and assign the frames.
