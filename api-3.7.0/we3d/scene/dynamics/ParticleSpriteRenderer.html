Classpublic class ParticleSpriteRenderer
InheritanceParticleSpriteRenderer we3d.scene.dynamics.ParticleRenderer

The ParticleSpriteRenderer renders a 2D (Flash) Sprite at every point. The Sprites are not scaled with 3D perspective. All clips have to be linked in the library to be able to be instanciated.
