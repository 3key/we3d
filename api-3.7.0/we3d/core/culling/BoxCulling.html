Classpublic class BoxCulling
InheritanceBoxCulling BoundBox we3d.core.culling.BoundSphere
Implements IObjectCulling

Default culling, bounding box object culling. The BoxCulling is set by default. It processes the object culling while rendering. If you add points directly to the points list of a SceneObject or if you animate points in a SceneObject then you have to update the bounding box. obj.objectCuller.testPoint( x,y,z ); // test a point, if its outside the boundig box, the box will be updated //or test a set of points obj.objectCuller.testPoints( obj.points ); If the Bounding Box is not up to date, the object may be culled to early (or in worst case never rendered) or to late.
