Property Detail
parentproperty
parent:Transform3d

The parent property can be set to a Transform3d Use null to set the root as the parent NOTE: Parents have to be created before childs in the parent hierarchy. If this is not possible, cause of async loading etc. you have to sort the objectList of the scene with parents first or use scene.utils.sortHierarchy If a child is before it's parent in the objectList, then the object matrix of the parent is not ready when processing the child (the matrix from the previous frame is taken)
