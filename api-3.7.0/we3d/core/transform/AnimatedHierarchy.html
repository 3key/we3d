addChannel()method
public function addChannel(tgt:Object, propName:String, uid:String, ec:IChannel):PropertyAnimator

Create a EnvelopeChannel in the transform. EnvelopeChannels are timelines for numeric properties. The property to animate don't have to be inside the object3d. 

		  var ec:EnbelopeChannel = new EnbelopeChannel();
			obj.transform.addChannel( _root, "myVar", "_root_myVar", ec);

Parameters

tgt:Object — The Target Object where the property is located
 
propName:String — The name of the property to animate
 
uid:String — Uniqe String ID for the new Envelope Channel, if a channel with the id is available it will be returned instead of creating a second one with the same id
 
ec:IChannel

Returns
PropertyAnimator — the new PropertyAnimator
