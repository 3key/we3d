Packagewe3d.view
Classpublic class View3d
InheritanceView3d flash.events.EventDispatcher

The View contains the rendered image. The View renders the Scene automatically if the Scene is invalid or every frame if forceUpdate is true. Use scene.invalidate() if something in the scene has changed. If autoRendering is disabled, you have to call the render method if you want to draw the scene. To display the rendered image, add the viewport somewere to the stage. The viewport contains all layers of the view. The width and height of the view will be inherited to Layers if their size is zero. Note that the size of the Camera in the scene is not resized. You should keep the same width and height in View3d and Camera3d.
