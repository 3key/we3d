Classpublic class EnvelopeChannel
InheritanceEnvelopeChannel LinearChannel Object
Implements IModifier

EnvelopeChannel provides numeric animation with keyframes. Every keyframe can have a unique easing method. Use the storeFrame method of the LinearChannel class to add keyframes to the channel: var ec:EnvelopeChannel = new EnvelopeChannel(); ec.storeFrame( 100, 3.1415 ); trace( ec.getValue(50) );
